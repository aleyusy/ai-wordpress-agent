# 🤖 AI WordPress Agent

**ИИ-агент для управления WordPress через чат.** Создавайте страницы, наполняйте их контентом, настраивайте дизайн, устанавливайте плагины и управляйте сайтом — просто общаясь с искусственным интеллектом.

---

## ✨ Возможности

### 🧠 AI-интеграция
- Поддержка **OpenRouter**, **OpenAI** и любых OpenAI-совместимых API
- Выбор модели из выпадающего списка (GPT-4o, Claude, Gemini, Llama и др.)
- Авто-загрузка доступных моделей через OpenRouter API
- Настраиваемая температура и лимит токенов

### 📄 Управление контентом
- Создание и редактирование страниц и постов
- Назначение шаблонов, родительских страниц, рубрик и меток
- Загрузка изображений по URL и установка миниатюр
- Управление категориями и тегами

### 🎨 Дизайн и внешний вид
- Переключение тем оформления
- Кастомный CSS (дополнение к теме)
- Настройка theme mods (цвета, шрифты и т.д.)
- Управление меню и их расположением

### 🔌 Плагины
- Установка плагинов из WordPress.org репозитория
- Активация и деактивация
- Просмотр установленных плагинов и их статуса

### 🧩 Виджеты и сайдбары
- Просмотр зарегистрированных сайдбаров
- Добавление и удаление виджетов
- Поддержка всех типов виджетов

### ⚙️ Настройки сайта
- Изменение любых опций WordPress
- Установка домашней страницы и страницы постов
- Управление пользователями и комментариями
- Просмотр полной информации о сайте

## 🚀 Установка

### Через GitHub (рекомендуется)

```bash
# Клонировать репозиторий в папку плагинов
cd /wp-content/plugins/
git clone https://github.com/ВАШ_ЛОГИН/ai-wordpress-agent.git

# Активировать плагин в админке WordPress
```

### Вручную

1. Скачайте [последний релиз](https://github.com/ВАШ_ЛОГИН/ai-wordpress-agent/releases)
2. Распакуйте архив в `/wp-content/plugins/ai-wordpress-agent/`
3. Активируйте плагин через меню **Плагины** в админке WordPress

---

## ⚡ Быстрый старт

1. Перейдите в **AI Agent → Настройки**
2. Укажите **API ключ** (получить можно на [OpenRouter.ai](https://openrouter.ai) или [platform.openai.com](https://platform.openai.com))
3. Выберите **модель** (по умолчанию `openai/gpt-4o-mini`)
4. Сохраните настройки
5. Откройте **AI Agent → Чат** и напишите:

> *«Создай страницу "О нас" с описанием компании, установи Contact Form 7 и добавь форму обратной связи на страницу»*

## 📋 Примеры запросов

| Запрос | Что произойдёт |
|--------|----------------|
| «Создай страницу "Услуги" и добавь три раздела с описанием» | Будет создана страница с контентом |
| «Установи плагин WooCommerce и активируй» | Плагин будет загружен и активирован |
| «Сделай страницу "Главная" домашней страницей» | Изменятся настройки чтения WordPress |
| «Добавь виджет поиска в сайдбар» | Виджет появится в боковой панели |
| «Поменяй тему на Twenty Twenty-Four» | Тема будет активирована |
| «Добавь кастомный CSS: header { background: #1a1a2e; color: white; }» | CSS будет добавлен в настройки темы |
| «Создай меню "Основное" и добавь туда все страницы» | Меню будет создано и заполнено |
| «Проверь, какие плагины требуют обновления» | AI проанализирует установленные плагины |

## 🔄 Обновления

Плагин поддерживает авто-обновления через **GitHub Releases**:

1. Укажите `ВАШ_ЛОГИН/ai-wordpress-agent` в настройках (**AI Agent → Настройки → GitHub репозиторий**)
2. При выходе новой версии на GitHub в админке WordPress появится уведомление об обновлении
3. Обновление в один клик — как с WordPress.org

## 🛠 Требования

- **WordPress** 5.8+
- **PHP** 7.4+
- **API ключ** OpenRouter, OpenAI или другого OpenAI-совместимого провайдера

---

## 🔒 Безопасность

- Все действия выполняются только после аутентификации в админке WordPress
- API ключ хранится в базе данных WordPress (опция `aiwp_settings`)
- Для AJAX-запросов используется nonce-проверка
- Доступ к чату имеют только пользователи с правом `manage_options`
